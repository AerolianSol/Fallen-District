# Fallen District

Fallen District is a terminal-based RPG game with a level-up system and strategic resource management. The game involves typing commands into the terminal to interact with the environment.

This program was initially developed on Replit in C++ before being ported to Github for ease of viewing.

# How to Run
In order to run the project, the user must open a terminal and navigate to the project folder. When inside the project folder, run ``main.cpp`` in order to start the program.
